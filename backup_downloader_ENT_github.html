Weekly ZIP Combiner Client App

This app fetches multiple ZIP files, renames each by appending its source domain (e.g., PH_DwC-A_lichenportal.zip), and bundles them into a single ZIP for you to download.

Your browser will prompt/save to your downloads folder.

Note: Cross-origin downloads require the source servers to allow CORS. If downloads fail due to CORS or you need automatic weekly scheduling, use the GitHub Actions workflow below.
