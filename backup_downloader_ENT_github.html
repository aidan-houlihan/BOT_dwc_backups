ANSP Weekly ZIP Combiner Fixed Sources

This tool downloads the two fixed DwC-A ZIP files, renames each by appending its source domain (e.g., ANSP-ENT_DwC-A_ecdysis.zip), and bundles them into a single ZIP you can save.

  • Midatlantic Herbaria
  • Bryophyte Portal
  • Mycportal
  • Lichen Portal
  • Macroalgae
Browser decides where to save (Downloads or prompt).

Tip: If downloads fail due to CORS or you need weekly automation, use the GitHub Actions workflow below. You can also right-click each source above and choose Save Link As… to download individually.
